# Klive Server

카페 소개 글 : https://cafe.naver.com/mk802

최초버전에서는 다른 웹서버와 같이 동작하지만, 0.2부터는 웹서버 모듈을 직접 사용하기에 웹서가 필요없고 ```python kliveProxy.py``` 로 실행하기를 권장합니다.

## 주요 업데이트
- Andorid App과 연동
- 채널을 커스텀 지원
- Everyon 버그 수정
- 각 계정정보 매일 새로 갱신하도록 변경
